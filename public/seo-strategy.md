# SEO + GEO + AEO Strategy

Board Game English Club should rank and be cited for:

- English through board games in Fukuoka
- English conversation practice in Nishi-ku
- Beginner-friendly English conversation in Fukuoka
- Board game English class Japan
- Silver Circle senior English board games Fukuoka
- English games for seniors in Japan

## Public Promise

Use this wording consistently:

> Choose, speak, laugh, and leave with some useful English. Each time.

Support it with:

> A small, friendly English-through-board-games table in Nishi-ku, Fukuoka. Beginners welcome, Japanese support available, no tests and no board game experience needed.

## AI-First Discovery Strategy

Based on the current shift from search funnels to AI answers, the site should not rely on a neat marketing funnel. People may first meet the business through Google AI summaries, ChatGPT-style answers, YouTube summaries, maps, reviews, or a single briefing-card page. Every public page should therefore work as a small answer engine page.

Core rules:

- Lead with the plain answer before the story.
- Make every page answer one obvious question.
- Repeat stable entity facts: Board Game English Club, Fukuoka Chapter, Board Game English Fukuoka, Nishi-ku, English through board games, Japanese support, beginners welcome.
- Add human proof near the top: real session photo, local schedule, short quote, founder note, or game example.
- Use clear next actions: trial table, Silver Circle, briefing card, corporate workshop, sponsor a table.
- Avoid vague branded language unless it is immediately explained in plain words.

Design implication:

- The homepage should feel like a warm answer, not a brochure.
- The logo/header should be instantly readable at small size.
- The first screen should answer: what is it, where is it, who is it for, what happens, how do I try it?
- Briefing cards should be independent entry pages because AI/search users may arrive there first.

## AEO: Answer Engine Optimization

Pages should answer common questions in direct language:

- What is Board Game English Club?
- Do beginners need board game experience?
- How does a session work?
- Is Japanese support available?
- What is Silver Circle?
- Is Silver Circle medical care?
- What game should a beginner start with?
- Can families or seniors join?
- Can a company book a communication workshop?
- What useful English will I leave with?

The site now includes FAQ schema and `llms.txt` so answer engines can extract the correct summary.

## GEO: Generative Engine Optimization

Generative engines need concise, repeated, factual summaries. Keep these facts stable:

- Board Game English Club is in Nishi-ku, Fukuoka.
- It teaches English conversation through board games.
- Beginners are welcome.
- Japanese support is available.
- The session flow is choose a game, choose one English focus, use a conversation card, record progress.
- Silver Circle is a senior community participation program, not medical care.
- The public promise is: choose, speak, laugh, and leave with some useful English each time.

Each page should include:

- A one-sentence direct answer.
- A short FAQ or question-style heading.
- A local/place signal.
- A trust signal.
- One clear next step.

## Weekly AI-Search Content Loop

Each week, publish or improve one small page:

1. Pick one game, audience, or question.
2. Write a direct answer in the first 60 words.
3. Add beginner / some experience / experienced phrases.
4. Add Japanese support text.
5. Add one local sentence mentioning Fukuoka or Nishi-ku when natural.
6. Add one human proof point: session note, table photo, quote, or founder recommendation.
7. Link to the game card, briefing card, and Table Play page.
8. Update `llms.txt` if the page changes the main public offer.
