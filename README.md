# Board Game English Club

Board Game English Club - Fukuoka Chapter is a Japanese-local, English-through-board-games website for families, seniors, adult learners, corporate workshops, and weekly briefing-card materials.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

The build generates static SEO/GEO/AEO doorway pages before creating the Vite production output.

## Weekly Content

```bash
npm run weekly:draft -- --game "Game Name"
```

Use the generated draft as the starting point for a weekly briefing/blog update.
